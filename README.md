# react-native-ui
Simple UI components based on existing ones or entirely new.

----

## Get Started

### Installation

Just run this `npm i react-native-ui --save`

----

### Usage

Start using the components.

```js
import { Button } from 'react-native-elements';

<Button />;
```
----
